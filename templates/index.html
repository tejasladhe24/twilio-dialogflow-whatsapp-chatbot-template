WhatsApp ChatBot

Made using Twilio-WA API and Dialogflow.

Start ChatBot in 3 steps


Create Twilio Account

Navigate to WhatsApp sandbox settings and add webhook url.

Create Dialogflow Agent

Create Service account on GCP and save key as "dialogflow.json".

Navigate to agent fullfilment settings and add the same webhook url.

Start Development

Fill values in settings.py file and run flask command to start local server
